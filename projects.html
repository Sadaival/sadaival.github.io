Robotics

ROS

This project include simulation of kortex robot (7 DOF) in Gazebo. The communication between various nodes is done through ROS (Robot Operating System). Codes are written for inverse kinematic and inverse dynamic controlers. The controller reads target end-effector positions and orientations. It makes a cubic-polynomial trajectory connecting these points and thereby does inverse kinematic calculation to output joint rotations for achieving the target pose.

Automatic gear Shifters

This project was implemented on Efficycle (tadpole configuration hybrid vehicle). Efficycle is powered by two people manually and also a DC Motor. Bot these manual and electric powers are added to power the vehicle on a single shaft. As the torques/speed delivered by the motor and human effort are different, seperate gear ratio is required for both of them. The gear ratio of both the motor and human effort shaft needs to be changed according to current speed. An Arduino MEGA was used as a controller, to implement speed calculation, gear control and display unit. The speed is calculated using a hall effect sensor, which calculates the time difference between two magnetic impulses recieved through magnets on the rim of back wheel. Through this speed input the arduino defines the effecient gear ratio and signals two servo motors to individually push derailleurs of gears placed on human effort and motor shafts. A seprate joystick module is also provided to control gears manually if needed. The current gear positions and speed is displayed on the display unit fitted on efficyle. This project was presented in SAE Efficycle 2018 held in Jalandhar and recieved the 'Best Innovation Award'. More details can be seen on the Github link.

Automobile

Formula Student

Formula Student is a global competition for engineering students to design racecars according to engineering specification provided. Our team participated in Formula Bharat 2020, and successfully designed and manufactured the vehicle. For this project I did the vehicle dynamics calculations for determining the initial parameters of the car and paticularly suspension subsystem. From this an iterative design process was used while incorporating all other subsystems with the suspension. Static and dynamic simulations, kinematic analysis was performed multiple times on suspension components. The suspension was manufactured to precision according to the final design and tested. As team captain I also worked in other subsystem design (steering, intake/exhaust, drivetrin, electronics, brake) and have a complete understanding of designing a formula student racecar.

Computer Vision

Melt pool monitoring

Exploring the realm of computer vision, this project was catered for capturing the molten pool during weld deposition in metal additive manufacturing. Specialised cameras were procured, and an experimental setup was designed for collection of data. effecientnet model was trained for capturing the melt pool size to be used for control of input weld parameters.

Additive Manufacturing

Toolpath generation

Traveling Salesman problem is an NP-hard optimisation problem in mathematics. But heurestic algorithms exists for solving it. Simialr heurestic algorithm of Lin-Kernighan was used to generate toolpath for use in additve manufacturing. The toolpath is of non-retractive nature and therefore has various benefits for WAAM (Wire arc additive manufacturing). A complete toolpath generation software was also made for this toolpath. the software took input of geometries and output G-codes for 3D printer. A part of this research work is published and further work on density based toolpath is in progress.

Powder simulation

Powder bed deopsition is a critical step for Powbed bed fusion based additive manufacturing. Current method of using Doctor blade for recoater puts additonal stress on already deposited layers. A non-contact deposition method is desirable and therefore this project focussed on simulating a vibration based non-contact powder recoating system. Discrete elemnent method simulations were performed at various vibration frequency and recoater geometries to determine the optimal parameters.
